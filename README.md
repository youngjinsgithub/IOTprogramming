## 📝 수업 개요

* [cite_start]**[01주차] 프로그래밍과 파이썬 소개** [cite: 2]
    * [cite_start]**프로그램이란?** 컴퓨터에게 일을 시키기 위해 자세한 명령어들을 순서대로 적어 놓은 문서(작업지시서)입니다[cite: 17, 18].
    * [cite_start]**프로그래밍을 왜 배울까?** 전공과 상관없이 업무 자동화 등 꼭 필요한 기술이 되었으며 [cite: 45][cite_start], 컴퓨터는 인간이 지루해하는 반복 작업을 아주 잘하기 때문입니다[cite: 57].
    * [cite_start]**프로그래밍 언어:** 컴퓨터가 이해할 수 있는 언어[cite: 66]. [cite_start]C, Java, Python 등이 있으며 [cite: 94][cite_start], '컴파일러'나 '인터프리터'가 우리가 작성한 코드를 기계어로 번역해줍니다[cite: 71].
    * [cite_start]**파이썬(Python)?** 1991년 귀도 반 로섬이 개발한 언어입니다[cite: 119].
        * [cite_start]**특징 1 (생산성):** C언어보다 훨씬 간결하고 효율적인 코드 작성이 가능합니다[cite: 122, 123, 124].
        * [cite_start]**특징 2 (인터프리터):** 명령을 입력하면 결과를 즉시 볼 수 있어 초보자에게 좋습니다[cite: 126, 127].
        * [cite_start]**특징 3 (풍부한 라이브러리):** 데이터 분석(pandas, numpy) [cite: 146, 150][cite_start], 머신러닝(scikit-learn) [cite: 140][cite_start], 웹(BeautifulSoup) [cite: 141] 등 강력한 라이브러리가 많습니다.
    * **개발 환경:**
        * [cite_start]**아나콘다(Anaconda):** 파이썬과 함께 numpy, pandas 등 인기 라이브러리가 대부분 포함된 배포판입니다[cite: 144].
        * [cite_start]**스파이더(Spyder):** 코드 편집기, IPython 콘솔, 디버거 등이 포함된 파이썬 통합 개발 도구(IDE)입니다[cite: 228, 234, 236].
        * [cite_start]**주피터 노트북 / Google Colab:** 코드와 설명을 함께 작성하며 실행할 수 있는 웹 기반 환경입니다[cite: 631, 746, 747].
    * **실행 모드:**
        * [cite_start]**대화형 모드(Interactive):** 콘솔에서 한 줄씩 바로 실행하고 결과를 확인합니다[cite: 343, 345].
        * [cite_start]**스크립트 모드(Script):** `.py` 파일에 코드를 작성한 뒤 한꺼번에 실행합니다[cite: 344, 350, 563]. [cite_start]코드가 복잡해지면 스크립트 모드를 사용합니다[cite: 358].
    * **기초 실습 (IPython 콘솔):**
        * [cite_start]**`print()` 함수:** 괄호 안의 문자열이나 계산 결과를 화면에 출력합니다[cite: 569, 584].
        * [cite_start]**문자열(String):** 큰따옴표(`"`)나 작은따옴표(`'`)로 감싼 텍스트 데이터입니다[cite: 326, 574].
        * [cite_start]**문자열 연산:** `+` (연결) [cite: 308][cite_start], `*` (반복)[cite: 318].
        * [cite_start]**터틀(Turtle) 그래픽:** 거북이를 움직여 화면에 그림을 그리는 기능입니다[cite: 766]. [cite_start](삼각형 그리기 실습 [cite: 816])
