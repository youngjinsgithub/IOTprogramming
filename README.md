# 🚀 < OOOO IoT 프로그래밍 > 수업 레파지토리

이곳은 < 202X년 X학기 OOO 교수님 >의 IoT 프로그래밍 수업 내용을 정리하고 실습 코드를 관리하기 위한 GitHub 저장소입니다.

## 📝 수업 개요

* **수업 목표:** 파이썬(Python) 언어를 기반으로 IoT 디바이스(라즈베리 파이, 아두이노 등)를 제어하고, 센서 데이터를 수집하며, 네트워크 통신을 통해 데이터를 송수신하는 프로그래밍 기술을 학습합니다.
* **주요 학습 언어:** Python
* **주요 하드웨어:** < 예: 라즈베리 파이 4, ESP32 >
* **주요 기술:** < 예: GPIO 제어, MQTT, HTTP 통신, JSON 데이터 처리 >
